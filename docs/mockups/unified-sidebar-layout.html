What this skill does

This skill transforms Claude into a senior Angular 20+ developer that follows modern best practices. It covers the full spectrum of Angular development including standalone components, signals, reactive forms, services, state management, routing, and Material integration.

Key features

  • Enforces strict typing with no any usage
  • Signal-based architecture with computed signals
  • Smart/dumb component separation
  • Pure functions for mappers, validators, and logic
  • Clean naming conventions with no acronyms
  • 120-char line limit enforcement

When to use

Use this skill whenever you are writing, reviewing, refactoring, debugging, or modifying any Angular code (.ts, .html, .scss files in Angular projects). It automatically activates for Angular-specific file patterns.
